Lab 3 - Countdown Timer

This is my Lab 3 project for COMP1073 Client-Side JavaScript.

The page displays a countdown from 10 to 0. When the countdown reaches 0, it displays Blastoff.

The background colour changes during the countdown:
- Yellow at 8
- Orange at 4
- Red at 0
